# Pomidoro Grand Prix

### Задание 1: CI/CD с Jenkins и Docker

**Цель:** Настроить автоматическую сборку Docker-образа через Jenkins Pipeline.

**Используемые файлы:**
- `Jenkinsfile` - конфигурация Jenkins Pipeline (автоматическая сборка образа)
- `Dockerfile` - определение Docker образа приложения
- `build.bat` - локальные скрипты для ручной сборки образа
  
---

### Задание 2: Развертывание в Minikube

**Цель:** Развернуть приложение в локальном Kubernetes кластере (Minikube).

**Используемые файлы:**
- `deployment.yaml` - конфигурация Kubernetes Deployment (поды приложения)
- `service.yaml` - конфигурация Kubernetes Service (доступ к приложению)
-  `deploy-minikube.bat`  - скрипт автоматического развертывания
- `check-status.bat`  - скрипт проверки статуса развертывания

---

### Задание 3: Мониторинг приложения

**Цель:** Настроить мониторинг приложения по 5 метрикам и создать дашборд.

**Метрики:**
1. **CPU** - использование процессора (в процентах)
2. **RAM** - использование памяти (в процентах)
3. **Disk** - использование диска (в процентах)
4. **Healthcheck** - статус здоровья приложения
5. **Business Metric** - бизнес-метрика (количество посещений страницы)

**Используемые файлы:**
- `collect-metrics.ps1`  - скрипт сбора метрик
- `create-dashboard.ps1` - скрипт создания HTML дашборда
- `dashboard.html` - HTML дашборд с метриками, который создается автоматически
- `metrics/` - директория с сохраненными метриками в JSON формате

---

## Основные файлы приложения

- `index.html` - главная страница приложения
- `script.js` - логика таймера Pomodoro
- `style.css` - стили приложения
- `components/` - компоненты приложения (navbar, footer)
